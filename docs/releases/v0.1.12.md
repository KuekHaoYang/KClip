# KClip v0.1.12

Scroller removal and editor headroom follow-up release.

## Highlights

- Removed the visible tray rail slider by suppressing every scroll view discovered inside the tray panel instead of only the first one.
- Gave edit mode a taller expanded panel height and clamped panel growth to the visible screen so the color editor no longer gets cut off.
- Hid the text editor scroll indicator directly to keep the editor surface free of slider chrome.
