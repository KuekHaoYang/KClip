# KClip v0.1.11

Color overlay layout follow-up release.

## Highlights

- Raised preview and edit overlays into an animated expanded panel so they sit above the tray instead of being clipped inside it.
- Kept the tray itself docked to the bottom edge, which stabilizes the color edit path when the lifted surface is clicked.
- Rebuilt `Color` swatches with explicit inset-radius math so the visible color block stays larger, cleaner, and free of redundant nested chrome.
