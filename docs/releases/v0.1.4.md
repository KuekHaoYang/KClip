# KClip v0.1.4

Code clip rendering release.

## Highlights

- Added rendered code blocks across tray cards, expanded preview, and edit mode, with per-clip language labels.
- Tightened automatic `Code` detection so mixed prose-plus-code clips stay untagged unless the entire clip is code, aside from comments.
- Preserved animated preview/edit transitions while fixing link suppression so hiding `Link` no longer exposes a bogus `Code` tag.
