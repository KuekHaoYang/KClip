# KClip v0.1.13

Bottom tray slider removal follow-up release.

## Highlights

- Removed the visible bottom tray slider by pinning the clip rail to a fixed card-height viewport with no spare scroller gutter.
- Added delayed scroll-view suppression retries after mount so late-appearing macOS scrollers also get stripped from the tray.
- Kept the horizontal rail on the explicit no-indicator scroll initializer as a second guard against the system reintroducing the slider.
