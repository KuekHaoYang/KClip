# KClip v0.1.16

Compact preview regression follow-up release.

## Highlights

- Restored text preview to the tray's resting panel height so preview no longer opens inside a tall mostly empty stage.
- Kept the taller expanded window for edit mode only, preserving the headroom needed for editor overlays.
- Added a regression test for preview panel height so preview and editor sizing cannot silently collapse back into one behavior.
