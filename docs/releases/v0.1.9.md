# KClip v0.1.9

Color clip layout follow-up release.

## Highlights

- Removed the remaining nested `Color` preview chrome so color cards now render as outer card -> color surface with no extra inner panel.
- Tightened the color preview overlay and editor sizing so the surface is visible without the panel feeling oversized.
- Kept animated color transitions while preserving the picker-sync crash fix from the previous patch.
