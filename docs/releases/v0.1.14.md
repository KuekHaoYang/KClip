# KClip v0.1.14

Tray top-edge clipping stabilization release.

## Highlights

- Added explicit resting headroom above the docked tray so the top border no longer gets clipped on some open animations.
- Kept the tray surface itself at the same visible height, avoiding another regression where the cards or color blocks would shrink.
- Locked the behavior behind a regression test and repeated auto-open window-size verification runs.
