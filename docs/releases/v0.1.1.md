# KClip v0.1.1

Image clipboard release.

## Highlights

- Added image clipboard history for copied photos, screenshots, and Finder image files.
- Fixed Finder image-file capture so previews use the real image content instead of the file icon.
- Added screenshot-folder monitoring for screenshot shortcuts that save files without copying to the clipboard.
- Tightened the tray card image preview into a full-width pill so compact media no longer reads as a square slab.
