# KClip v0.1.5

Code tagging follow-up release.

## Highlights

- Fixed copied SwiftUI snippets so property wrappers, access-controlled declarations, and modifier chains are still recognized as `Code`.
- Recomputed derived tags while loading saved clips so plain-text items misclassified in the previous build no longer reopen as `Code`.
- Reduced the expanded code preview footprint and removed the duplicated header badge so code previews read tighter and cleaner.

## Validation

- `swift test`
- `./script/build_and_run.sh --verify`
- `./script/make_release.sh v0.1.5`
