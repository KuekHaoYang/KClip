# KClip v0.1.6

Syntax-colored code rendering release.

## Highlights

- Added syntax-colored rendering for code previews so keywords, types, strings, and modifiers no longer appear as a flat white block.
- Preserved animated rendered-code transitions while moving the preview body to a dedicated highlighter instead of plain text.
- Kept the tighter code preview footprint from the previous release.
