# KClip v0.1.2

Image preview polish release.

## Highlights

- Removed the redundant in-preview `Image` badge so media blocks show only the image itself.
- Added explicit fade-and-scale transitions when image previews appear or swap content.
- Kept the rounded inset formula for expanded previews so the visible image corners match the container padding.
