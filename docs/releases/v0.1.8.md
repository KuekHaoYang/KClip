# KClip v0.1.8

Color clip follow-up release.

## Highlights

- Simplified `Color` previews to a single inner palette surface so the preview no longer shows an extra nested block or overhangs the bottom edge.
- Fixed the color editor quit path by stopping picker sync from rewriting the clip while the editor is opening or when the selected swatch changes.
- Kept animated preview and editor transitions while tightening the color editor controls to a simpler menu-plus-picker row.
