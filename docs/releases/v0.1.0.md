# KClip v0.1.0

Initial public release.

## Highlights

- Fast bottom-edge tray interaction from the menu bar.
- Local-only text clipboard history.
- Search, tags, pinning, preview, edit, delete, and drag export.
- Release packaging, CI, and public repo documentation.
