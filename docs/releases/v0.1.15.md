# KClip v0.1.15

Image card and filtered selection follow-up release.

## Highlights

- Removed the nested inner frame from compact image clips so the card reads as one preview surface instead of a surface inside another surface.
- Fixed tag-filtered tabs to match `All`: first click selects, second click pastes, with the armed paste state cleared when the visible clip set changes.
- Preserved animated image transitions while adding regression coverage for both the image-card layout and the filtered-tab activation flow.
