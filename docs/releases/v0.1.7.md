# KClip v0.1.7

Color clip release.

## Highlights

- Added first-class `Color` clips for standalone hex codes and palette-style lists, with real swatch previews in the tray and expanded overlay.
- Tightened `Color` detection so prose that happens to mention a hex value no longer gets tagged as a color clip.
- Added animated color editing with a palette-backed `ColorPicker` while keeping direct hex-code editing in the text field.
